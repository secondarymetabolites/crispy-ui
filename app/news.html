2016-05-02Version 1.1.0 of CRISPy-web released.
We just released version 1.1.0 of CRISPy-web with some bug fixes and improvements based on user feedback.

Highlights

  • All results are being shown.
    The previous version did a stricter filtering for sgRNA quality. This turned out to limit the usefullness in some use cases. We now always show all results, sorted from least to most off-target hits.
  • Easier to localise sgRNAs.
    For sgRNAs that are located within an annotated ORF, the identifier of the ORF is now also shown. This should make it easier to keep track of which sgRNA belongs to which ORF.
  • More robust sequence parser.
    Previously, CRISPy-web would sometimes crash when certain invalid sequence files were uploaded, sometimes causing outages for all CRISPy-web users. The new version should be much more robust to this, improving the overall user experience.
